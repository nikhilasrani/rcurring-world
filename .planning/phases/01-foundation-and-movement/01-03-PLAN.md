**Step 3: Create `scripts/extrude-tilesets.sh`:**
```bash
#!/bin/bash
# Re-extrude all tilesets from raw sources
# Run this whenever raw tileset PNGs are updated
set -e

TILE_W=16
TILE_H=16
RAW_DIR="./raw-tilesets"
OUT_DIR="./public/assets/tilesets"

for tileset in ground buildings nature decorations; do
  echo "Extruding $tileset..."
  npx tile-extruder \
    --tileWidth $TILE_W \
    --tileHeight $TILE_H \
    --input "$RAW_DIR/$tileset.png" \
    --output "$OUT_DIR/$tileset.png"
done

echo "All tilesets extruded successfully."
```
Make it executable: `chmod +x scripts/extrude-tilesets.sh`

Player spawn: tile coordinates approximately (45, 35) -- at the MG Road Metro station exit.

**Tilemap JSON structure (Tiled format):**
The JSON must be a valid Tiled JSON export format with:
- `width`: 60, `height`: 60, `tilewidth`: 16, `tileheight`: 16
- `orientation`: "orthogonal"
- `renderorder`: "right-down"
- `tilesets` array referencing the four extruded tilesets with correct `firstgid`, `tilewidth: 16`, `tileheight: 16`, `margin: 1`, `spacing: 2` (extrusion parameters)
- `layers` array with these layers in order:

  1. **ground** (tile layer): Fill entire map. Roads as asphalt tiles, sidewalks along roads, grass in parks, paths through Cubbon Park.
  2. **ground-detail** (tile layer): Road markings, crosswalk stripes, cracks, puddles. Sparse -- only where it adds texture.
  3. **buildings** (tile layer): Building walls for all 5 landmarks plus generic shops/buildings along MG Road. Multi-tile structures using building tileset.
  4. **above-player** (tile layer): Tree canopies over walkable paths, roof overhangs, bridge rails. Player walks UNDER these tiles.
  5. **collision** (tile layer): Invisible layer. Mark all building interiors, walls, water, dense vegetation, and map edge barriers as blocked. These tiles have the `ge_collide: true` property in the tileset definition. Leave roads, sidewalks, paths, and open areas clear.
  6. **spawn-points** (object layer): Objects with properties:
     - `{ name: "player-spawn", x: 720, y: 560, type: "spawn" }` (pixel coords = tile coords * 16)
  7. **zones** (object layer): Rectangle objects defining landmark areas:
     - `{ name: "chinnaswamy-stadium", x, y, width, height, type: "landmark" }`
     - Same for ub-city, cubbon-park, vidhana-soudha, mg-road-metro

**Tileset properties for collision:**
In the tilesets array, each tileset that has blocking tiles must include a `tileproperties` or `tiles` object mapping tile IDs to `{ "ge_collide": true }`. Specifically:
- All building wall tiles: `ge_collide: true`
- Water tiles: `ge_collide: true`
- Dense tree trunk tiles: `ge_collide: true`
- Boundary fence/wall tiles: `ge_collide: true`

**Map edge boundaries:**
Row 0 and row 59: thick trees/fences (collision)
Column 0 and column 59: thick trees/fences (collision)

**Dense decoration (Pokemon-like):**
- Place street vendor tiles, auto-rickshaws, lampposts, bus stops along MG Road
- Place benches, flowers, trees in Cubbon Park
- Place metro pillars and signs near MG Road Metro station
- No empty grass or road tile should be left unadorned if a nearby decoration makes sense
