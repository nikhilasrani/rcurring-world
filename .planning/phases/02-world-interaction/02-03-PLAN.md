**src/entities/NPC.ts:**

```typescript
export class NPC {
  public sprite: Phaser.GameObjects.Sprite;
  public def: NPCDef;
  private scene: Phaser.Scene;
```

- Constructor takes `(scene: Phaser.Scene, def: NPCDef)`, creates sprite at (0,0) via `scene.add.sprite(0, 0, def.spriteKey, 1)`, sets depth to 2, calls `registerAnimations(def.spriteKey)`.
- `registerAnimations(spriteKey)`: Same pattern as Player.ts. Creates walk-down/left/right/up and idle-down/left/right/up animations with the same frame layout (3 cols x 4 rows). Walk frameRate=8, idle frameRate=2. Uses `scene.anims.create()` with key format `${spriteKey}-${dirKey}`. Check `anims.exists()` before creating to avoid duplicates.
- `getGridEngineCharacterConfig()`: Returns `{ id: def.id, sprite, walkingAnimationMapping: 0, startPosition: def.position, speed: def.speed, facingDirection: def.facing, offsetY: -4, collides: true }`.
- `destroy()`: Calls `sprite.destroy()`.

**src/systems/NPCManager.ts:**

export class NPCManager {

  private npcDefs: Map<string, NPCDef> = new Map();

- `spawnNPC(scene: Phaser.Scene, gridEngine: any, npcDef: NPCDef)`: Creates `new NPC(scene, npcDef)`, stores in `npcs` Map keyed by npcDef.id, stores def in `npcDefs` Map. - `getAllNPCIds(): string[]`: Returns Array.from(npcs.keys()).
- `destroy()`: Calls destroy on all NPC instances, clears maps.

**src/systems/InteractionSystem.ts:**

```typescript
export class InteractionSystem {
  private signDefs: SignDef[] = [];
  private interiorDefs: InteriorDef[] = [];
```

- Constructor takes `(signDefs: SignDef[], interiorDefs: InteriorDef[])`. Stores them.
- `checkInteraction(gridEngine: any, playerCharId: string): InteractionTarget | null`:
  1. Get `facingPos = gridEngine.getFacingPosition(playerCharId)`.
  2. Check `gridEngine.getCharactersAt(facingPos, undefined)` for NPC IDs (any char that is not `playerCharId`). If found, return `{ type: 'npc', id: npcId, position: facingPos }`.
  3. Check `signDefs` for a sign at `facingPos.x, facingPos.y`. If found, return `{ type: 'sign', id: sign.id, position: facingPos }`.
  4. Check `interiorDefs` for a door at `facingPos.x, facingPos.y` (matching `doorPosition`). If found, return `{ type: 'door', id: interior.id, position: facingPos }`.
  5. Return null.
- `getSignData(signId: string): SignDef | undefined`: Find in signDefs.
- `getInteriorData(interiorId: string): InteriorDef | undefined`: Find in interiorDefs.

**src/ui/DialogBox.ts:**

```typescript
import TextTyping from 'phaser3-rex-plugins/plugins/behaviors/texttyping/TextTyping';
export class DialogBox {
```

- Constructor takes `(scene: Phaser.Scene)`. Creates all game objects as children of the scene. All positioned in the UIScene's camera coordinate space (not world space). Uses `setScrollFactor(0)` on all elements.
- **Background:** `scene.add.graphics()`. `fillStyle(0xF8F8F8, 0.95)`, `fillRect(4, 244, 472, 72)`. `lineStyle(2, 0x222222, 1)`, `strokeRect(4, 244, 472, 72)`. Set depth 60.
- **NPC Name text:** `scene.add.text(12, 248, '', { fontFamily: 'monospace', fontSize: '12px', color: '#333333', fontStyle: 'bold' })`. Depth 61.
- **Content text:** `scene.add.text(12, 264, '', { fontFamily: 'monospace', fontSize: '10px', color: '#111111', wordWrap: { width: 456 } })`. Depth 61.
- **Page indicator:** `scene.add.text(460, 304, '', { fontFamily: 'monospace', fontSize: '10px', color: '#666666' })`. Shows unicode down-triangle when more pages exist. Depth 61.
- **TextTyping:** `new TextTyping(this.contentText, { speed: 30 })`. Listen to `typing.on('complete', () => { this.isTypingComplete = true; })`.
- All elements start hidden via `setVisible(false)`.

Methods:
- `show(dialogue: DialogueData)`: Sets pages array, currentPage=0, sets nameText (empty string if no name), calls `typePage(0)`, sets all elements visible, emits `EVENTS.DIALOGUE_OPEN` via eventsCenter.
- `advance()`: If `!isTypingComplete` -> `typing.stop(true)` and set `isTypingComplete=true`. Else if more pages -> increment currentPage, call `typePage(currentPage)`. Else -> call `hide()`, emit `EVENTS.DIALOGUE_CLOSE`.
- `typePage(pageIndex: number)`: Sets `isTypingComplete=false`, calls `this.typing.start(this.pages[pageIndex])`. Updates page indicator: show down-triangle if `pageIndex < pages.length - 1`, hide otherwise.
- `hide()`: Set all elements not visible.
- `isActive(): boolean`: Returns whether dialogue box is currently visible.
- `destroy()`: Destroy all game objects.

IMPORTANT: Do NOT call `setInteractive()` on any dialogue box element. The advance input is handled by the WorldScene/UIScene input handler, not by clicking on the dialogue box itself.
