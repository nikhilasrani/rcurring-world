**Step B: Interior tilemap generation script** (`scripts/generate-interior-tilemaps.js`)

Follow the pattern from `scripts/generate-tilemap.js`. Generate 4 Tiled-format JSON tilemaps.

Each interior tilemap must have these layers (same naming convention as outdoor map):
- `ground` (tile layer): floor tiles
- `buildings` (tile layer): walls, furniture, counters, shelves
- `above-player` (tile layer): ceiling details (optional, can be empty)
- `collision` (tile layer): invisible blocking tiles on walls and furniture edges
- `spawn-points` (object layer): single object with name="player-spawn", x/y in pixel coords
- `zones` (object layer): single zone rect covering entire interior with zone name
- `doors` (object layer): single object for exit door with name="exit", plus properties: `returnZone`, `returnX`, `returnY`

**Metro station** (15x10): Gray floor, gray walls, ticket counter (row of counter tiles), benches, metro map poster on north wall, platform edge tiles along south side, turnstile near entrance. Spawn point near south door (x=7, y=8 in tiles -> x=112, y=128 in pixels).

**Coffee shop** (12x10): Wood floor, brick walls, counter along north wall with steaming cups, 3 tables (2x1 tile each) with chairs, menu board on east wall, welcome mat at entrance. Spawn at (x=5, y=8 -> x=80, y=128 pixels).

**UB City mall** (15x12): Marble floor with checkered pattern, white walls, decorative escalator (escalator tiles on west side), shop front display cases along walls, potted plants, ceiling lights. Spawn at (x=7, y=10 -> x=112, y=160 pixels).

**Cubbon Park library** (12x10): Dark wood floor, wood panel walls, bookshelves covering north and east walls, 2 reading tables with chairs, photo frames on south wall, potted plant in corner. Spawn at (x=5, y=8 -> x=80, y=128 pixels).

The tilemap JSON format must match this structure:
```json
{
  "compressionlevel": -1,
  "height": H,
  "width": W,
  "tilewidth": 16,
  "tileheight": 16,
  "orientation": "orthogonal",
  "renderorder": "right-down",
  "tiledversion": "1.10.2",
  "type": "map",
  "version": "1.10",
  "tilesets": [{ "firstgid": 1, "source": "", "name": "interior", "image": "../tilesets/interior.png", "imagewidth": 128, "imageheight": 128, "tilewidth": 16, "tileheight": 16, "tilecount": 64, "columns": 8 }],
  "layers": [...]
}
```

Note: For the tileset reference in the tilemap JSON, use inline tileset definition (not external .tsx file) with `"image"` pointing to the relative path `"../tilesets/interior.png"`.

Output to `public/assets/tilemaps/`:
- interior-metro.json
- interior-coffee.json
- interior-ubcity.json
- interior-library.json

Both scripts should be runnable standalone and idempotent.
