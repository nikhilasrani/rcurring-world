**src/ui/ZoneBanner.ts:**

```typescript
import Phaser from 'phaser';
import { GAME_WIDTH } from '../utils/constants';

export class ZoneBanner {
  private container: Phaser.GameObjects.Container;
  private background: Phaser.GameObjects.Graphics;
  private bannerText: Phaser.GameObjects.Text;
  private scene: Phaser.Scene;
  private isShowing = false;
```

- Constructor takes `(scene: Phaser.Scene)`. Creates a Container at y=-32 (off-screen above), adds Graphics background and Text to it.
- **Background:** `scene.add.graphics()`. `fillStyle(0x111111, 0.7)`. `fillRect(0, 0, 480, 32)`. Full width of game.
- **Text:** `scene.add.text(240, 16, '', { fontFamily: 'monospace', fontSize: '12px', color: '#ffffff', fontStyle: 'bold' })`. Origin (0.5, 0.5) for centering.
- Container depth = 50. Container visible = false. Container y = -32.
- `setScrollFactor(0)` on the container so it stays fixed on screen.

Methods:
- `show(zoneName: string)`: If `isShowing` is true, return early (don't stack banners). Set `isShowing = true`. Set bannerText to zoneName. Set container visible. Animate with tweens:
  1. Slide in: `scene.tweens.add({ targets: container, y: 8, duration: 300, ease: 'Power2' })`.
  2. On complete: `scene.time.delayedCall(2000, () => { ... })` for hold time.
  3. After hold: Slide out: `scene.tweens.add({ targets: container, y: -32, duration: 300, ease: 'Power2', onComplete: () => { container.setVisible(false); isShowing = false; } })`.
- `destroy()`: Kill all tweens, destroy container.

IMPORTANT: Do NOT call `setInteractive()` on any banner element. The banner must be purely visual and non-blocking per CONTEXT.md locked decision.

**src/systems/ZoneManager.ts:**

```typescript
import { eventsCenter } from '../utils/EventsCenter';
import { EVENTS, TILE_SIZE } from '../utils/constants';
import type { LandmarkDef, DiscoveryState } from '../utils/types';

export class ZoneManager {
  private currentZone: string | null = null;
  private discoveredZones: Set<string> = new Set();
  private discoveredLandmarks: Set<string> = new Set();
  private landmarks: LandmarkDef[] = [];
```

- Constructor takes `(landmarks: LandmarkDef[])`. Stores landmarks array.
- `checkZone(playerTilePos: { x: number; y: number })`: For each landmark, check if `playerTilePos.x >= landmark.position.x && playerTilePos.x < landmark.position.x + landmark.size.width && playerTilePos.y >= landmark.position.y && playerTilePos.y < landmark.position.y + landmark.size.height`. Note: Zone positions in mg-road.json are already in tile coordinates (not pixel coordinates). If player is in a landmark zone and it's different from `currentZone`: set `currentZone = landmark.id`, add to `discoveredZones` and `discoveredLandmarks`, emit `eventsCenter.emit(EVENTS.ZONE_ENTER, landmark.name)`. If player is not in any zone: set `currentZone = null` (do NOT emit zone-enter for leaving a zone).
- `enterInterior(displayName: string)`: Emit `EVENTS.ZONE_ENTER` with the interior's displayName. Track interior zone in discoveredZones.
- `getDiscoveryState(): DiscoveryState`: Returns `{ discoveredZones: Array.from(discoveredZones), discoveredLandmarks: Array.from(discoveredLandmarks), currentZone, currentInterior: null }`.
- `isDiscovered(landmarkId: string): boolean`: Check discoveredLandmarks set.
- `getDiscoveredCount(): number`: Return discoveredLandmarks.size.
- `getTotalLandmarks(): number`: Return landmarks.length.

**src/systems/TransitionManager.ts:**

```typescript
import Phaser from 'phaser';
import { eventsCenter } from '../utils/EventsCenter';
import { EVENTS, ASSETS, LAYERS, TILE_SIZE, COLLISION_PROPERTY } from '../utils/constants';
import type { InteriorDef } from '../utils/types';

export class TransitionManager {
  private scene: Phaser.Scene;
  private isTransitioning = false;
  private currentInterior: InteriorDef | null = null;
  private outdoorState: {
    tilemapKey: string;
    returnPosition: { x: number; y: number };
  } | null = null;
```

- Constructor takes `(scene: Phaser.Scene)`.
- `enterBuilding(interiorDef: InteriorDef)`: If `isTransitioning`, return. Set `isTransitioning = true`. Store `currentInterior = interiorDef`. Save outdoor return state: `outdoorState = { tilemapKey: ASSETS.TILEMAP_MG_ROAD, returnPosition: interiorDef.returnPosition }`.
  1. Emit `EVENTS.MOVEMENT_FREEZE` with `true` to freeze player.
  2. `scene.cameras.main.fade(250, 0, 0, 0)`.
  3. Listen `camera.once('camerafadeoutcomplete', () => { ... })`:
     a. Restart WorldScene with interior data: `scene.scene.restart({ mode: 'interior', interiorKey: interiorDef.tilemapKey, tilesetKey: interiorDef.tilesetKey, playerSpawn: interiorDef.playerSpawn, interiorSize: interiorDef.size, interiorId: interiorDef.id, interiorDisplayName: interiorDef.displayName, returnPosition: interiorDef.returnPosition, returnTilemapKey: ASSETS.TILEMAP_MG_ROAD })`.
     Note: WorldScene.init() will need to handle this data in Plan 05. For now, TransitionManager just prepares and triggers the restart.

- `exitBuilding()`: If `isTransitioning` or `!outdoorState`, return. Set `isTransitioning = true`.
  1. Emit `EVENTS.MOVEMENT_FREEZE` with `true`.
  2. `scene.cameras.main.fade(250, 0, 0, 0)`.
  3. On fade complete: Restart WorldScene with outdoor data: `scene.scene.restart({ mode: 'outdoor', playerSpawn: outdoorState.returnPosition })`.

- `isInInterior(): boolean`: Returns `currentInterior !== null`.
- `getCurrentInterior(): InteriorDef | null`: Returns currentInterior.
- `setTransitionComplete()`: Set `isTransitioning = false`. Called by WorldScene after scene restart completes. This is the handshake point.
- `resetToOutdoor()`: Clear currentInterior, clear outdoorState.

The actual scene restart handling (rebuilding tilemap, reinitializing Grid Engine) will be wired in Plan 05 when WorldScene.init() is modified. TransitionManager only orchestrates the fade + restart trigger.
